#  使用されていない Windows Phone 8.0 ポータル サイト アプリ

2016 年 9 月の時点で、Windows 8.0 ポータル サイト アプリは使用されていません。 登録されている Window 8.0 デバイスは管理されますが、新しいデバイスは登録できません。
