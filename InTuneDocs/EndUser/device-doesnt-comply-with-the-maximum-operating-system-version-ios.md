# iOS デバイスが最新のオペレーティング システムのバージョンに準拠していない

お使いの iOS デバイスは、最新バージョンのオペレーティング システムについての IT 管理者のポリシーに準拠していません。 IT 管理者にお問い合わせください。 連絡先情報については、[ポータル サイト Web サイト](http://portal.manage.microsoft.com)をご確認ください。
