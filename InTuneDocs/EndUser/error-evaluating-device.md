# デバイスの評価エラー
デバイスのいずれかに準拠していないことが検出されましたまたはこの状態のデバイスを評価する際、会社のポリシーと、エラーが発生しました。

この問題を解決するには、数分待ってからし、**[ポリシー準拠状況の確認]** をタップします。.

問題が解決しない場合は、IT 管理者に依頼します。
